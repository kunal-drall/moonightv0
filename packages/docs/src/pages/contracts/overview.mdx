# Smart Contracts Overview

Moonight Protocol consists of 11 Cairo smart contracts deployed on Starknet. All contracts use OpenZeppelin components for access control, pausability, and reentrancy protection.

## Core Contracts

| Contract | Purpose | Key OZ Components |
|----------|---------|-------------------|
| [CDPManager](/contracts/cdp-manager) | CDP lifecycle, liquidations, interest | Ownable, Pausable, ReentrancyGuard |
| [MoonUSD](/contracts/moonusd) | ERC-20 stablecoin with role-based mint/burn | ERC20, Ownable, Pausable |
| [PriceOracle](/contracts/price-oracle) | Pragma wrapper with TWAP + fallback | Ownable |
| [StabilityPool](/contracts/stability-pool) | Liquidation absorption pool | Ownable, Pausable, ReentrancyGuard |
| [RedemptionManager](/contracts/redemption-manager) | Sorted queue + redemption fee decay | Ownable |
| PositionNFT | ERC-721 CDP ownership token | ERC721, Ownable |
| ProtocolConfig | Global parameters registry | Ownable |

## Vault Contracts

| Contract | Strategy | Details |
|----------|----------|---------|
| [Vault A](/vaults/vault-a) | Delta-neutral funding rate | Long/short BTC perp |
| [Vault B](/vaults/vault-b) | Leveraged BTC accumulation | Recursive CDP loop |
| [Vault C](/vaults/vault-c) | ERC-4626 yield optimizer | SP + Ekubo + LZ |
| [Vault D](/vaults/vault-d) | Covered call + PT/YT | Options + yield splitting |

## External Interfaces

| Interface | Protocol | Usage |
|-----------|----------|-------|
| `IPragmaABI` | Pragma Oracle | BTC/USD price feeds |
| `IEkuboRouter` | Ekubo DEX | Token swaps, LP |
| `IExtendedDex` | Extended | Margin trading, funding rates |
